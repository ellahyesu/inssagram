# Spring Project - inssagram(SNS)
## Description
+ 개인 프로젝트(1인)
+ 개발 기간
  - 2021.08.11 ~ 2021.08.30
+ 개발 환경
  - Java, HTML, CSS, JavaScript
  - jQuery, Bootstrap
  - Spring Boot 4.11, Apache Tomcat 9.0, Mybatis, MySQL
  - MVC Pattern, Ajax
  - Kakao oven
  - Git
+ 역할
  - 프로젝트 일정 계획
  - UI 기획
    * 프로토타이핑 툴 : 카카오 오븐 사용 https://ovenapp.io/project/jH0ei1yrTzqqEruubKbA3uttvsGtt9Eq#cjcMh 
  - DB 설계, API 설계
  - 프로젝트 개발환경 구축
  - 구현 
    * 프론트 엔드 - 모든 페이지 구현 
    * 백 엔드
      1. 회원가입 기능
      2. 로그인 기능
      3. 프로필 이미지 변경 기능
      4. 타임라인 글쓰기 기능
      5. 댓글 쓰기 기능
      6. 좋아요 기능
      7. 팔로우 기능
      8. 페이징 기능
  - GitHub 버전, 이슈(트러블슈팅 등) 관리
  - 동작 테스트
